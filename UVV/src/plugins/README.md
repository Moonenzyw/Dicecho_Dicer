## 插件一览

```python
common #不是插件，不做解析，提供一些共有的访问和函数
--- roll #公用骰点机制，基于官方骰娘插件魔改
--- db #本地sqlite数据库机制，用于暂存log，缓存用户的人物卡等

nocard #提供和人物卡无关的骰点解析，包括基于官方骰娘插件的.ti，.li解析以及.rXdY，.rb，.rp等解析
logger #基于官方log插件的log记录器，解析.log相关
card #用户人物卡管理，.sc， .ra，.en检定和解析（如果有人物卡的话，进行对应人物卡的检定）
     #.st数值设定和人物卡导入； .cocX人物卡创建
helper #.help解析
```
